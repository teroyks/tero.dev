This site runs on:

- [Eleventy](https://www.11ty.dev/) (v3) — static site generator
- [GitHub Pages](https://docs.github.com/en/pages) — hosting
- [JSON Feed](https://www.jsonfeed.org/) — syndication
- [just](https://just.systems/) — command runner
- [Markdown](https://commonmark.org/) — content authoring
- [Nunjucks](https://mozilla.github.io/nunjucks/) — templating
- [Pa11y](https://pa11y.org) — accessibility helper
- [Pico CSS](https://picocss.com/) — lightweight CSS framework
- [Prism](https://prismjs.com/) — syntax highlighting for code snippets

The site ships zero client-side JavaScript.
Light and dark modes follow your system preference.
